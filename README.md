## Getting Started

1. Clone the repo

   ```sh
   git clone https://github.com/ShawnAlisson/thePersonal.git
   ```

2. Install NPM packages

   ```sh
   npm i
   ```

3. Run

   ```sh
   npm start
   ```

4. Open [http://localhost:3000] with your browser to see the result.
